## 修复历史任务创建时间问题

**问题**：任务创建时未设置 `created_at` 字段，导致查询时显示错误的创建时间。

**修复方案**：

修改 `sqlite_repository.go` 中的 `CreateTask` 方法，在 INSERT 语句中添加 `created_at` 字段：

```go
// 修改前
query := `INSERT INTO gift_code_task (code, all_done) VALUES (?, 0)`

// 修改后  
query := `INSERT INTO gift_code_task (code, all_done, created_at) VALUES (?, 0, CURRENT_TIMESTAMP)`
```

- [sqlite_repository.go](file:///Users/penitence/Code/renjie/cdk-get/internal/storage/sqlite_repository.go#L326) - 修改 INSERT 语句
